# 📊 Procurement Analytics Dashboard (Power BI)

## 🚀 Project Overview  
This project presents an interactive **Procurement Analytics Dashboard** built using **Power BI**.  
It provides deep insights into procurement spending, supplier performance, budget control, and risk optimization.

The dashboard helps organizations improve **cost efficiency, supplier performance, and decision-making** using data-driven insights.

---

## 📌 Key Features  

### 📈 Executive Overview
- Total Procurement Spend: **31.83M**  
- Total Orders: **161**  
- On-Time Delivery: **56.5%**  
- Total Budget: **5.93M**  
- Potential Savings (10%): **3.18M**

## ▶️ How to Use  

1. Download the `.pbix` file  
2. Open it in Power BI Desktop  
3. Refresh the dataset if required  
4. Use filters (Year, Supplier Type, Product Category)  
5. Explore insights across different dashboard pages  

---

## 📊 Key Insights  

- Procurement spend is highest in **EPC category**  
- On-time delivery is below optimal (~56%), indicating improvement scope  
- Certain suppliers show **high risk with lower performance**  
- Significant **budget overspending** observed  
- Potential savings opportunities up to **10%**
